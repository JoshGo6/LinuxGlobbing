In Linux, you can match filename patterns using regex or globbing. While regex is more powerful, it's also more difficult to use, its implementation varies across different tools, and not every tool supports it. Globbing is more universally available across tools, and it can often provide the functionality you need considerably more easily. 

This introduction to globbing shows you the basics of using globbing patterns, with an emphasis on the `**` pattern, since I personally found that one somewhat more difficult to master. This repo was built based on my own research into the topic so that I and others could benefit from a one-stop shop to understand the basics of globbing and the `**` globbing pattern, in particular.
